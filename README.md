# nette-project

This is a simple application on [Nette Framework](https://doc.nette.org/en/2.4/)

## Installation
+ run ```chmod -R a+rw temp log``` in the project folder
+ run command ``` composer install```
  
## Requirement
+ PHP 7.1
+ Nette Framefork
+ Doctrine ORM
+ MYSQL 5.6
